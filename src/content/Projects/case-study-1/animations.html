Web Animations API

Slide Me

Implementation: Use `element.animate()` with keyframes and timing options.

Show Code 
document.getElementById('wa-slide').animate([
{ transform: 'translateX(0px)', opacity:1 },
{ transform: 'translateX(200px)', opacity:0.3 },
{ transform: 'translateX(0px)', opacity:1 }
], { duration:2000, iterations:1 });
